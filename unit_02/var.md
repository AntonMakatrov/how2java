# Ключевое слово var

Одним из самых интересных нововведений Java 10 безусловно является вывод типа локальной переменной ([JEP 286](http://openjdk.java.net/jeps/286)).
Это дает возможность сократить объявления переменных используя новое ключевое слово `var`:

```java
public class Main {
    public static void main(String[] args) {
        var users = new ArrayList<User>();
        var x = 10;
    }
}
```

Слово var ставится вместо типа данных, а сам тип переменной выводится из того значения, которое ей присваивается.
Например, переменой `x` присваивается число **10**, значит, переменная будет представлять тип `int`.

Если переменная объявляется с помощью `var`, то мы _обязательно_ должны инициализировать ее,
то есть предоставить ей начальное значение, иначе мы получим ошибку, как, например, в следующем случае:

```java
public class Main {
    public static void main(String[] args) {
        var x;      // Ошибка! переменная не инициализирована
        x = 10;
    }
}
```

Индексы для циклов также могут быть объявлены с помощью var:

```java
public class Main {
    public static void main(String[] args) {
        var numbers = List.of("a", "b", "c");

        for (var v : numbers) {
            System.out.print(v + " ");
        }

        for (var i = 0; i < numbers.size(); i++)
            System.out.print(numbers.get(i) + " ");
    }
}
```

При объявлении локальных переменных вы можете использовать `var` вместо имени класса или интерфейса, чтобы позволить компилятору вывести тип.
Тип, выводимый компилятором, помещается в байт-код, поэтому во время выполнения ничего не меняется — Java все еще является статически типизированным языком.
