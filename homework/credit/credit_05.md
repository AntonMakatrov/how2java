# Задание на зачет №5

Введите число с клавиатуры. Определите, является ли это число [простым](https://ru.wikipedia.org/wiki/Простое_число)


Пример:

```java
Введите число: 3
Число 3 простое
```

```java
Введите число: 7
Число 7 простое
```

```java
Введите число: 15
Число 15 не простое
```

```java
Введите число: 19
Число 19 простое
```

```java
Введите число: 39
Число 39 не простое
```

```java
Введите число: 47
Число 47 простое
```
